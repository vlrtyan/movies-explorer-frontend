# Проект "Movies Explorer" | frontend-часть

## Это сервис, в котором можно найти фильмы по запросу и сохранить в личном кабинете

## Технологии

* HTML
* CSS
* JS
* React
* Express.js
* MongoDB

## Инструкции по запуску

1. `npm i`

2. `npm start`
